# 🔎 OpenRecon

OpenRecon is a lightweight OSINT reconnaissance framework written in Python.

It performs automated discovery and extraction of:

- Links
- Emails
- Phone numbers
- IP addresses

It also classifies sensitive data and supports JSON export.

---

## 🚀 Features

- Live structured terminal output
- Data classification (info / critical)
- Bing search integration
- Automatic HTML fetching
- JSON export option
- VS Code debug ready
- Modular architecture
- GitHub-ready structure
